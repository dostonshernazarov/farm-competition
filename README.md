# farmish-crm


<h2>Clone project</h2>
```
git clone https://github.com/dostonshernazarov/farm-competition.git
```
<h2>Create .env file. </h2>
<h3>Example: <a href="./.env">.example.env</a> </h3>

<h2>How to run</h2>

```
make migrate-up
```

**1. With make file** <br>

```
make run
```

**1. With terminal** <br>

```
go run cmd/app/main.go
```
